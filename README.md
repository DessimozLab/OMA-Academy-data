## OMA Academy data
[OMA](https://omabrowser.org/) is a knowledgebase that entails several aspects of comparative genomics, including a pre-computed orthology database, proteome quality assessement, fast orthology inference tools, synteny and functional analyses tools, etc.
In this repo, you can find the data required for running the tutorials of several OMA resources found in the [OMA Academy](https://omabrowser.org/oma/academy/). OMA Academy is an e-learning website with a suite of self-learning modules specifically designed to help computational biologists use OMA. Each module is stand-alone, with a basic introduction and a series of exercises with hints and answers.

```
source /workspace/conda/bin/activate
conda activate omacademy
```
